# Retail Sales Analytics Dashboard

## Project Overview

This project analyzes 4 years of retail sales data using Power BI and Excel to identify trends, profitability patterns, and business insights.

## Tools Used

- Power BI
- Microsoft Excel

## Key Features

- KPI Tracking
- Sales Analysis
- Profit Analysis
- Region-wise Insights
- Category-wise Insights
- Interactive Filters and Slicers

## Dataset

Superstore Dataset (10,000+ sales records)

## Skills Demonstrated

- Data Analysis
- Data Visualization
- Dashboard Development
- Power BI
- Excel
